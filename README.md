# Cinema APP

Full-stack web application for managing and browsing cinema content. The project includes a Vue.js frontend, Express.js backend, and a database connected through a REST API. Users can view available movies, check showtimes, and interact with cinema data through a modern web interface. The backend handles API requests, business logic, and database operations, while the frontend provides a responsive user interface.

Main features include:

REST API built with Express.js

Frontend built with Vue.js

Database integration for storing movies and showtimes

CRUD operations for managing cinema data

Separation of frontend and backend architecture

The project demonstrates full-stack development concepts such as API design, database integration, and client-server communication.
